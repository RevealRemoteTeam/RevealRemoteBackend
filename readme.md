# RevealRemoteBackend
## Files
- **index.js** entry point of the application
- **env.js** environment variables
- **modules** parts of the application
	+ **client.js** model for a Client object
	+ **const.js** constants
	+ **state.js** model for a State object
	+ **validation.js** input validator
- **www** exposed static files
	+ **presentation.js** script that will be injected into presentations by a bookmarklet
